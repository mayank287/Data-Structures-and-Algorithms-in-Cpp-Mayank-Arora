------------------------ Self Balancing Binary Search Tree ------------------------
The Idea is to do Some restructuring (or re-balancing) when doing insertion/deletion

Example - 1
Insert 100 200 and 300 in Empty BST

100  ---->  100  ----> 100            ------->    200  (Left Rotate)
               \          \                      /   \ 
               200        200                  100   300
                            \
                            300



300  ---->  300  ----> 300        ------->     200  (Right Rotate)
            /          /                      /   \ 
           200        200                   100   300
                     /
                   100      
                            300



Rotation is O(1) Operation

Type of Self Balancing Bianry Search Tree
1> AVL Trees
2> Red Black Tree
